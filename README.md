# Pulse Analysis

This project is dedicated to the analysis of vehicle crash test pulse data.

## Core Responsibilities

-   Processing raw TDMS signal data.
-   Applying filters and performing numerical integration to calculate velocity and displacement.
-   Calculating various performance metrics from the processed signals.
-   Visualizing and comparing analysis results.

## Usage

1.  Install dependencies: `pip install -r requirements.txt`
2.  Place the `nhtsa_data.db` file in the `data` directory.
3.  Run analysis scripts from the `scripts` directory.
